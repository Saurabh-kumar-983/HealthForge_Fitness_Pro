# HealthForge Fitness Pro

Ready-to-publish static website package.

## Included
- Rebranded premium UI: HealthForge Fitness
- 6 body-part exercise pages
- 66+ exercise cards
- Step-by-step execution guidance
- Safety points on every exercise
- Health tools page
- Admin page for daily health posts/blogs using browser localStorage
- Responsive design for mobile, tablet and desktop

## Publish
Upload all files in this folder to GitHub Pages, Netlify, Vercel static hosting, or any normal web server.

## Media note
The current GIF files are preserved in `assets/gifs/`. Replace those files with real human exercise GIFs/videos using the same names to make every demo realistic without changing HTML.

## Admin note
The current admin panel is static and saves posts in the browser. For multi-user production publishing, connect the same form to Supabase/Firebase/your backend.
