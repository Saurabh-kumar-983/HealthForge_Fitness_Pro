01. Pull-Up

Benefit: Back width, grip, and upper-body strength.

Steps to perform
  1. Start with a neutral spine and engaged lats.
  2. Pull using the back, not only the arms.
  3. Pause briefly at peak contraction.
  4. Return slowly without rounding the lower back.
Safety points
  • Warm up first and start with a light set.
  • Stop if you feel sharp pain, dizziness, or joint discomfort.
  • Keep breathing, avoid holding your breath for long periods.

09. Reverse Pec Deck

Benefit: Rear delts and upper back.

Steps to perform
  1. Set up with balanced stance and correct equipment height.
  2. Brace your core and align the joints before moving.
  3. Perform each repetition slowly with full control.
  4. Return to the start position without losing posture.
Safety points
